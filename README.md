# CodeAlpha_DataAnalytics_Tasks
# 📊 Data Analytics Project (Web Scraping + EDA + Data Visualization)

## 🧾 Overview
This repository contains a complete end-to-end data analytics project covering:

- Web Scraping (Data Collection)
- Exploratory Data Analysis (EDA)
- Data Visualization (Python + Tableau)

The project demonstrates how raw web data is collected, cleaned, analyzed, and transformed into meaningful insights using visual storytelling.

---

# 🧩 Project Structure

This project is divided into 3 tasks:

- ✅ Task 1: Web Scraping  
- ✅ Task 2: Exploratory Data Analysis (EDA)  
- ✅ Task 3: Data Visualization  

---

# 🌐 TASK 1: Web Scraping

## 📌 Objective
To extract structured data from websites using Python and build custom datasets for analysis.

## 🛠 Tools Used
- Python
- BeautifulSoup
- Requests
- Pandas

## 📊 Key Activities
- Extracted data from web pages
- Parsed HTML structure
- Collected relevant datasets
- Cleaned and stored data in CSV format

## 📌 Skills Learned
- HTML parsing
- Web navigation structure
- Data extraction techniques
- Dataset creation from scratch

# 📊 TASK 2: Exploratory Data Analysis (EDA)

## 📌 Objective
To understand data structure, patterns, trends, and anomalies using statistical and visual methods.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Key Activities
- Data cleaning and preprocessing
- Statistical summary analysis
- Trend identification
- Correlation analysis
- Outlier detection

## 📈 EDA Visualizations
- Monthly Sales Trend 📈
- Top Selling Products 🏆
- Sales Distribution by Region 🌍
- Sales vs Profit Analysis 💰
- Heatmaps and Boxplots

## 📌 Insights
- Sales show seasonal trends
- Few products dominate revenue
- Regional performance varies significantly
- Profit does not always align with sales

# 📊 TASK 3: Data Visualization

## 📌 Objective
To transform raw data into meaningful visual insights using charts, graphs, and dashboards.

## 🛠 Tools Used
- Matplotlib
- Seaborn
- Tableau

## 📊 Visualizations Created
- KPI Cards (Total Sales, Total Profit)
- Sales Trend Over Time
- Region-wise Sales Analysis
- Sales vs Profit Scatter Plot
- Product Performance Analysis
- Box Plot for Distribution

## 📈 Dashboard Features
- Interactive filters (Region, Product, Date)
- Dynamic visual exploration
- Business insights for decision-making
- Professional dashboard design

## 📌 Key Insights
- Strong seasonal sales patterns
- Few products contribute most revenue
- Some regions consistently outperform others
- Profitability varies across products

# 📊 Sales Analysis & Visualization Project (Python + Tableau)

## 🧾 Overview
This project focuses on analyzing sales data using Python (Jupyter Notebook) and creating an interactive dashboard using Tableau.

The goal is to extract meaningful insights from raw sales data using data visualization techniques.

---

## 📂 Dataset Information
The dataset contains the following columns:

- Order Date – Date of each transaction  
- Region – Sales region  
- Product – Product name  
- Sales – Revenue generated  
- Profit – Profit earned  

---

## 🛠 Tools & Technologies Used
- Python 🐍  
- Jupyter Notebook 📓  
- Pandas 📑  
- Matplotlib 📊  
- Seaborn 🎨  
- Tableau 📈  

---

# 📊 Exploratory Data Analysis (Python)

### 🔹 1. Monthly Sales Trend 📈
- Sales grouped by Order Date (month-wise)
- Line chart used to show sales trend over time
- Helps identify seasonal patterns and growth trends

---

### 🔹 2. Top Selling Products 🏆
- Products ranked based on total sales
- Bar chart used to visualize top-performing products
- Helps identify key revenue drivers

---

### 🔹 3. Sales Distribution by Region
- Pie chart using Matplotlib
- Shows contribution of each region to total sales

---

### 🔹 4. Sales vs Profit Analysis
- Scatter plot using Seaborn
- Shows relationship between sales and profit

---

### 🔹 5. Region-Month Heatmap
- Pivot table created using Pandas
- Heatmap using Seaborn
- Shows monthly regional performance

---

### 🔹 6. Sales Distribution by Region (Box Plot)
- Box plot used to show distribution and outliers
- Compares sales variation across regions

---

## 📈 Tableau Dashboard

### 📌 Visualizations Included:
- Monthly Sales Trend

- Sales vs Profit Scatter Plot
- Top Products Analysis
- Box Plot Analysis

---

### 📌 Features:
- Interactive filters (Region, Product, Date)
- Dynamic dashboard
- Easy comparison of performance
- Business insights for decision-making

---

## 🔍 Key Insights

- Sales show strong monthly seasonal trends 📈  
- A small number of products contribute most of the revenue  
- Regional performance varies significantly 🌍  
- Profit is not always proportional to sales 💰  
- Some regions perform consistently better than others

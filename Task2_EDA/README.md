# Exploratory Data Analysis (EDA) - Titanic Dataset

## Project Overview
This project is part of the CodeAlpha Data Analytics Internship. The objective is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify trends, patterns, and insights using statistical analysis and visualizations.

## Dataset
Titanic Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objectives
- Explore the dataset structure and data types.
- Identify missing values and anomalies.
- Analyze trends and patterns.
- Create visualizations to support findings.

## Visualizations
- Histogram (Age Distribution)
- Bar Chart (Gender Distribution)
- Box Plot (Fare Distribution)

## Key Findings
- Female passengers had a higher survival rate than male passengers.
- Passenger ages varied across different groups.
- Fare values contained several outliers.
- Missing values were present in some columns.

## Conclusion
The Exploratory Data Analysis helped in understanding passenger demographics, fare distribution, and survival patterns in the Titanic dataset through statistical summaries and visualizations.

# Task 2: Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis on the Titanic dataset to identify patterns, trends, and relationships in the data.

## Dataset
Titanic Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- SciPy

## Analysis Performed
- Data Structure Exploration
- Missing Value Detection
- Trend and Pattern Identification
- Hypothesis Testing
- Anomaly Detection

## Visualizations
- Survival Distribution (Bar Chart)
- Gender vs Survival (Bar Chart)
- Passenger Class Distribution (Bar Chart)
- Age Distribution (Histogram)
- Fare Distribution (Histogram)

## Key Findings
- Female passengers had higher survival rates.
- First-class passengers had better survival chances.
- Most passengers belonged to third class.
- Missing values were handled before analysis.
